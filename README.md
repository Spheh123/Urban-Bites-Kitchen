# Urban Bites Kitchen

Luxury restaurant landing page and ordering experience built with plain HTML, CSS, and JavaScript.

## Included files

- `index.html`: page structure and content
- `style.css`: dark luxury design system, responsive layout, animations
- `script.js`: editable menu data, filters, cart UI, WhatsApp ordering
- `netlify.toml`: static hosting configuration for Netlify

## Quick edit points

- Change menu items in `script.js` inside the `menuItems` array.
- Change the WhatsApp destination number in `script.js` by updating `restaurantPhone`.
- Update contact details and section copy in `index.html`.
- Adjust colors and spacing in `style.css` under the `:root` variables.

## Local preview

Open `index.html` directly in a browser, or use a lightweight local server if you prefer.

## Deployment

This project can be uploaded directly to GitHub and deployed on Netlify as a static site.
